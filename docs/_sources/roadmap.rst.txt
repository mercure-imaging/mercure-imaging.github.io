Development Roadmap
===================

Planned
-------

The following features are planned for implementation in the next mercure version(s).

* UI option for reprocessing discarded cases
* Processing priorities for rules (Normal, Priority, Off-Peak)


Completed
---------

* "Force discard" option for rules
* Configurable webhooks on reception/dispatch of series
* Optional processing modules (executed prior to dispatching), allowing mercure to be used also as DICOM processing platform
