Discussion Board
================

If you have questions or feature requests for mercure, you can get in touch with the development team through GitHub Discussions. You need to sign in using a GitHub account before you are able to post.

`mercure Support Board <https://github.com/orgs/mercure-imaging/discussions>`_

Please submit bug reports directly to the `Issues tracker on Github <https://github.com/mercure-imaging/mercure/issues>`_.
